Figma currently supports `shadows` `typography`* `colors` 
`icons` and `logos`. Tokens like `motion` and `spacing` 
are not supported out of the box. 

***

### Detaching type styles
There will be instances where you need to mix styles such as a _italics_ & ~~strikethrough~~ with normal text only then should you 
detach from our text tokens in figma. We reccomend that you select the token that is the closest to what you need then begin to modify
the text. Be sure when you are adjusting line height you still reference our [line-height tokens](/tokens/our-tokens/typography).

### How to properly change icons
When swapping icons be sure not to double click the icon but rather select the icon and use the component

## Using Spacing & Motion
### Setting Spacing
Since figma does not support enforcing spacing tokens it will be required by designer that all spacing in
the designs are using our included spacing tokens. To set the value select and input the token size value. 
Zeplin hand off tools do include a check to see if the spacing is one of our tokens. Designers should make it 
a habit to check using zeplin to quickly find spacing discrepancies
