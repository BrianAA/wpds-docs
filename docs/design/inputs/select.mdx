## Usage guidelines 
<!-- Start of a guide -->
<Guide guidance="don't">
   <div style={{zIndex:5, width:'150px'}}><SelectExample/></div>
</Guide>

#### Overflow Text
Size the select input according to the data you will be taking on. Since the input currently does not change in height 
input data will need to be designed around to avoid ugly overflow.
<br/>

<!-- Start of a guide -->
<Guide guidance="do">
   <div style={{zIndex:5, width:'250px'}}><MonthExample/></div>
</Guide>

#### Grouping 
Use for grouping items to save space.
<br/>

<!-- Start of a guide -->
<Guide guidance="do">
   <div style={{zIndex:5, width:'250px'}}><PinkThings/></div>
</Guide>

#### Grouping 
Labels in select inputs are very important as the options may 
read as different from each other without a label.

## Specs


<Container>
  <img width="60%"  height="auto" src='/img/design/input/select/specs.svg'/>
</Container>
<div className="mt-sm mb-lg">

##### Size
Select input has a fixed height of 48px and the width takes 100% of its 
parent container. The height when options are revealed is 100% of the total amount of options.
</div>

<Container>
  <img width="50%"  height="auto" src='/img/design/input/select/spacing.svg'/>
</Container>
<div className="mt-sm mb-lg">

##### Spacing
Spacing of the select input is 8px padding top and bottom, 8px left, 16px right and 4px between the input and the 
error message.
</div>
