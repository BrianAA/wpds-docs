## Usage guidance
<!-- Start of a guide -->
<Guide guidance="caution">
  <div style={{width:100}}>
<Input
  id="boop"
  labelText="State"
  name="State"
  onBlur={function noRefCheck(){}}

  type="text"
  value="MD"
/>
</div>

</Guide>

#### Avoid small widths
If a small width text input is required be sure the expected input will be short.  
<br/>


<Guide guidance="don't">
      <div className="mr-sm" style={{width:150}}>
<Input
  id="boop"
  labelText="City"
  name="City"
  onBlur={function noRefCheck(){}}
  onFocus={function noRefCheck(){}}
  size="small"
  type="text"
/>
</div>
  <div style={{width:100}}>
<Input
  id="boop"
  labelText="State"
  name="State"
  onBlur={function noRefCheck(){}}
  onFocus={function noRefCheck(){}}
  size="default"
  type="text"
  value="MD"
/>
</div>

</Guide>

#### Vary input heights
If more than one input is being used all inputs must be the same size variant.   
<br/>
