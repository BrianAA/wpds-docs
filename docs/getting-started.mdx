# Design

Before using tokens in Figma, you need to be sure you are added to Figma org. To be added, 
please be sure to submit a SAR (service as request) through service now in your [okta apps](https://washpost.okta.com/) 
for figma as a editor.

After being approved and added, you will want to make sure you are familiar with how we work in Figma. Go through the presentation
[_How we work_](https://www.figma.com/proto/g4b0zZXMoWzsJm5icnYWSB/ðŸ’¼-How-we-work?page-id=0%3A1&node-id=1%3A2&viewport=287%2C48%2C0.05&scaling=contain&starting-point-node-id=1%3A2) 
to onboard and be ready to work in figma.
